## EaseImageSave
A google chrome extension to save images from a opened webpage as file with three select able image formats: JPG, PNG, WEBP. 

### Problem (Core Idea)
Users want to save an image easily for later use while visiting a webpage. In some cases the user cant save images using the default browser context menu, cause the image is containered in a image slider or gallery. They also cant selected a file format for the image.

### Solution
A google chrome extension which can be activated by rightclicking over an image or an image in a slider or gallery. Over a submenu the user can select the desired image format for saving. After selecting the image format a default file browser window open, where the user can define the target directory, where to save the file.
